Bardzo prosta gra typu Roguelik napisana w C++ przy użyciu biblioteki Raylib.
Było kompilowane na linuxie przy użyciu standardowego GCC.
Jest zawarty makefile, który dobrze działał pod Arch Linuxem.
Gra się wyłącza po przegrania, tak ma działać.

Linki do biblioteki:
https://github.com/raysan5/raylib/wiki/Working-on-GNU-Linux
